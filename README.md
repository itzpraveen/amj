# AMJ Qatar Website

Static website for Al Majid Jawad W.L.L, developed for `amjqatar.me`.

## Local preview

Open `index.html` directly in a browser, or serve the folder locally:

```sh
python3 -m http.server 4173 --bind 127.0.0.1
```

Then visit `http://127.0.0.1:4173/`.

## Brand logo assets

Original portfolio logo assets collected from `https://amjqatar.me/brands/` are stored in `assets/brand-logos/`. The source URL and category metadata are tracked in `assets/brand-logos/manifest.json`.
